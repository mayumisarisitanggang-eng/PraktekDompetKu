## ✅ Fitur Aplikasi

- Menampilkan saldo terkini secara otomatis
- Mencatat transaksi pemasukan dan pengeluaran
- Riwayat transaksi menggunakan `FlatList`
- Nominal berwarna **hijau** untuk pemasukan, **merah** untuk pengeluaran
- Fitur **Reset** untuk menghapus semua data dan mengembalikan saldo ke Rp 0
